# Product Scope

## Version 1.0 (MVP)
### Goal
Build a secure, offline-first password manager with all essential features.

### Included Features

- Authentication
- Master Password
- Create/Edit/Delete Login
- Folder
- Tag
- Search
- Password Generator
- Password Strength Checker
- Clipboard
- Trash
- Attachments
- Metadata
- Local Encrypted Vault

Professional productivity features.
